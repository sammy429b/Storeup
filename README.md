## Clone the repository:

```bash
git clone https://github.com/sammy429b/Storeup.git
```

## Navigate to the project directory:

```bash
cd Storeup
```

## Install dependencies:

```bash
npm install
```
or
```bash
yarn install
```

## Start the development server:

```bash
npm run dev
```
or

```bash
yarn run dev
```

# Open the app in your browser:
```bash
http://localhost:5173/
```
